# Teste Técnico - Target Sistemas

Este repositório contém o teste técnico realizado para a vaga de estágio na Target Sistemas.

## Descrição do Projeto

O projeto consiste em um sistema que implementa diversas funcionalidades para análise de dados e manipulação de strings. As principais funcionalidades são:

1. **Cálculo de Soma**:
   - Calcula a soma dos primeiros 13 números naturais utilizando um laço `while`.

2. **Verificação de Fibonacci**:
   - Verifica se um número fornecido pelo usuário pertence à sequência de Fibonacci.

3. **Análise de Faturamento Diário**:
   - Lê e desserializa um arquivo JSON com dados de faturamento diário.
   - Calcula e exibe:
     - Menor valor de faturamento diário.
     - Maior valor de faturamento diário.
     - Média mensal de faturamento.
     - Número de dias com faturamento superior à média.

4. **Percentual de Representação por Estado**:
   - Calcula e exibe o percentual de representação de faturamento por estado, baseado em valores predefinidos.

5. **Inversão de String**:
   - Recebe uma string do usuário e exibe a versão invertida dessa string.
